Enter a variant below to see its SpliceAI scores. This can be any SNV or simple InDel within an exon or intron defined by GENCODE v36.

more details...
This web interface and the underlying SpliceAI server are being developed by the TGG at the Broad Institue.
The SpliceAI server optionally uses pre-computed scores provided by Illumina for SNVs and small InDels, but falls back on running the SpliceAI model directly for scores not available in the pre-computed files - such as larger InDels or non-default "Max distance" values.
All code is available at github.com/broadinstitute/SpliceAI-lookup.

For more information on SpliceAI, see Jaganathan et al, Cell 2019 and github.com/Illumina/SpliceAI, as well as in this recorded talk by Kishore Jaganathan.
